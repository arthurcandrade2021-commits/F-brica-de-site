Atividade de Fixação

Atividade do Arthur Correia
  1. As manipuladores de eventos no JavaScript são basicamente
    movimento com o mouse, um carregamento de uma págino
    por exemplo. Só que o principal propósito é ter uma página
    interativa e ter uma certa "diversão" nas páginas web.

  2. Esse processo de adição de manipulação é como se fosse um
    botão, por exemplo um input, que adiciona uma barra que
    você pode digitar, dentre outras.

  3. Usa-se o método addEventListener() para adicionar um
    manipulador de eventos via JavaScript, como em
    botao.addEventListener("click", funcao). Essa abordagem
    evita misturar HTML com JavaScript e torna o código mais
    organizado e fácil de manter.

  4. Ao clicar em uma div, você passa por um processo de captura
    e bolha. A captura, é desce. Enquanto a bolha sobe. Pois,
    é bom saber esse estado de bolha, e de caputura em um
    DOM (document).

  5. Delegação é um elemento-pai que pode ser usado para alterar a
    si mesmo e, junto modifiva os filhos usando o addEventListener,
    para alteração. A vantagem é um desempenho melhor e um código
    reduzido, e diminuído a repetição.
